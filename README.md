# Hangman 🎯

A console based Hangman game built in C#.

## Features
- Random word selection
- Letter by letter guessing
- 6 lives per game
- Reveals correctly guessed letters
- Detects win and loss conditions

## How to play
- Run the program
- A secret word will be chosen randomly
- Guess one letter at a time
- You have 6 lives
- Guess the word before you run out of lives!

## Built with
- C#
